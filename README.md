qBOT-alpha
==========
arduino software to drive cometlab project named qBOT

OSC message system
------------------

# from qBOT (port 7001):
* /knock 
* /ping
* /face
* /ip

# from qBOT (port 9001):
* /wek/inputs

# to qBOT (port 8000):
* /led
* /ledpwm
* /knockthresh
* /knockdelay
* /sendip
* /ping
* /ledrange
* /configdata
* /imudelay
